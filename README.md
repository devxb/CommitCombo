<br> - <del>깃허브가 캐시한 이미지가 깃허브 서버에 남아있는거로 보입니다. (해결책을 생각중입니다!)</del>

</li>

<br> - <del>[Ctrl + shift + R] 로 애니메이션을 강제 시작할수있습니다.</del>
